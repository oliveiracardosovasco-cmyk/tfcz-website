## Nebenwirtschaft / Ausschank / Bewilligungen

**Aktueller Stand:**
- Vereinslokal, kein öffentlicher Gastrobetrieb
- Bar im Rahmen von Vereinsaktivitäten
- Flaschengetränke, kein Gewinnzweck im klassischen Gastro-Sinn

**Unterscheide sauber:**

| Modell | Beschreibung | Rechtliches Risiko |
|---|---|---|
| Heute (intern) | Bar für Vereinsmitglieder, Events | Tief |
| Temporäre Eventlösung | Ausschank an grossen Publikumsturnieren | Mittel (Bewilligung prüfen) |
| Organisatorische Ausweitung | Gastro für externe Besucher | Hoch (Gastrokonzession?) |
| Stärkere Kommerzialisierung | Öffentliche Bar, Gewinnzweck | Sehr hoch |

> ⚠️ Rechtliche Fragen zu Bewilligungen, Lebensmittelrecht, Gewerbebewilligung müssen vor Ausweitung geklärt werden.

## Offene Fragen Clubentwicklung

- Wieviele aktive Mitglieder hat der TFCZ aktuell?
- Was ist das Jahresbudget des Vereins?
- Welche Positionen im Vorstand existieren?
- Gibt es eine Mitgliedschaftsgebühr?
- Wie viele Helfer stehen regelmässig zur Verfügung?
- Wo liegt die grösste Kapazitätsgrenze (Lokal? Helfer? Budget?)?

> Diese Fragen müssen beantwortet werden, bevor konkrete Professionalisierungsschritte geplant werden.
