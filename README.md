# TFCZ — Tischfussball Club Zürich (Website)

Moderne, animierte Vereins-Website für den **Tischfussball Club Zürich (TFCZ)**.
Single-File HTML/CSS/JS nach dem TFCZ-Brand- & Design-System (Nunito Sans, Blau/Gold/Navy,
Brand-Line blau oben / gold unten, Kicker-Signature).

## Dateien

| Datei | Zweck |
|---|---|
| `index.html` | Öffentliche Website (Auslieferung) |
| `index-cms.html` | Gleiche Website **mit eingebautem CMS** (Texte/Bilder/Links/Abstände direkt im Browser bearbeiten, HTML-Export, Backup) |
| `assets/img/` | Logo, Wappen-Pattern, optimierte Event-Fotos |

## CMS benutzen (`index-cms.html`)

Im Browser öffnen → Button **„ Bearbeiten"** (unten rechts) → Panel öffnet sich:
Texte/Bilder/Links anklicken, Einträge mit ＋/⧉/, Design-Regler, Sektionen ein-/ausblenden.
**Speichern** legt die Änderungen im Browser ab; **„Fertige HTML exportieren"** gibt eine
eigenständige Datei aus.

## Hosting

Statische Seite — kann direkt auf GitHub Pages, Netlify o.ä. gehostet werden.
Für GitHub Pages: `index.html` als Einstieg, Pfade sind relativ.
